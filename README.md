# App React PokeAPI
Link APP: https://pokemon-api-reactv.netlify.app/

![image](https://github.com/FreddyArreagaM/app-pokeApi/assets/127709400/91ab1bde-bf8d-48ae-92a3-58d2558ea1c3)

El objetivo de este proyecto es crear una web mostrando los datos de la PokeAPI utilizando React y CSS.

Requisitos:

- Mostrar 20 Pokémons
- Paginación 
- Implementar 5 filtros de activación única o de acumulativos.
